Solucion preguntas capitulo 1:

1.  La estructura de una red involucra una combinación de dispositivos, medios de comunicación 
    y servicios que trabajan juntos para permitir la transferencia eficiente de datos y la 
    comunicación exitosa entre usuarios y sistemas. Aquí hay una descripción general de los 
    componentes clave:

    Dispositivos de red:

        Los dispositivos en una red se pueden clasificar en dos categorías principales: 
        dispositivos finales (o dispositivos de usuario) y dispositivos intermedios 
        (o dispositivos de red). Cada categoría cumple un papel específico en el proceso 
        de comunicación en red.

        Dispositivos Finales (Dispositivos de Usuario):
            Estos son los dispositivos utilizados por los usuarios finales para acceder a la 
            red y realizar diversas actividades. Los dispositivos finales interactúan directamente 
            con los usuarios y con los servicios y recursos de la red. Algunos ejemplos de dispositivos 
            finales son: estaciones de trabajo (Computadoras de Escritorio y Portátiles),dispositivos 
            móviles, impresoras y escáneres, teléfonos VoIP y cámaras de seguridad.
        Dispositivos Intermedios (Dispositivos de Red):
            Estos dispositivos trabajan en conjunto para enrutar, conmutar y gestionar el
            flujo de datos a través de la red. Desempeñan un papel crucial en la
            transferencia de datos entre los dispositivos finales y aseguran que los datos
            lleguen a su destino correcto. Algunos ejemplos de dispositivos intermedios
            son: enrutadores, conmutadores (Switches), puntos de acceso inalámbrico y
            firewalls.

    Medios de Comunicación:

        - Cableado Estructurado: Cable de cobre o fibra óptica utilizado para conectar dispositivos dentro de la red. Puede ser Ethernet para redes cableadas.
        - Redes Inalámbricas: Utilizan señales de radio o infrarrojas para permitir la Comunicación 
          sin cables. Ejemplos incluyen Wi-Fi y Bluetooth.

    Servicios y Protocolos:

        Los servicios y protocolos son elementos fundamentales en una red, ya que permiten la 
        comunicación, el intercambio de datos y el funcionamiento eficiente de los dispositivos 
        y aplicaciones en el entorno de red. Aquí hay una descripción de algunos servicios y 
        protocolos clave en una red

        - Protocolos de Comunicación: Conjuntos de reglas y estándares que gobiernan cómo los 
          dispositivos se comunican entre sí. Ejemplos incluyen TCP/IP, HTTP, SMTP.
        - Direccionamiento IP: Cada dispositivo en la red tiene una dirección IP única, 
          lo que permite la identificación y la ruta adecuada para la comunicación.
        - Servicios de Nombres de Dominio (DNS): Traducen nombres de dominio legibles por humanos 
          en direcciones IP numéricas.
        - Servicios de Resolución de Nombres (DHCP): Asigna direcciones IP dinámicamente a dispositivos 
          en la red para facilitar la configuración automática.
        - Servicios de Seguridad: Incluyen firewalls, sistemas de detección y prevención de 
          intrusiones (IDS/IPS), VPN para proteger la red contra amenazas cibernéticas.
        - Servicios de Calidad del Servicio (QoS): Prioriza y gestiona el tráfico en función 
          de la importancia y los requisitos de ancho de banda de las aplicaciones.

2.  RED:
    Una red es un conjunto de dispositivos interconectados que pueden comunicarse y compartir 
    recursos, puede ser de diferentes tamaños y alcances, desde pequeñas redes domésticas hasta
    grandes redes empresariales o globales.

    LAN (Red de Área Local):
    Una LAN es una red de tamaño limitado que abarca un área geográfica pequeña, como una casa, 
    una oficina o un edificio, las LAN generalmente utilizan tecnologías de cableado estructurado 
    o conexiones inalámbricas para conectar dispositivos dentro del área.

    WAN (Red de Área Amplia):
    Una WAN es una red que abarca un área geográfica más amplia, como una ciudad, un país o 
    incluso a nivel global, las WAN a menudo se construyen utilizando tecnologías como líneas 
    telefónicas, fibra óptica o conexiones inalámbricas, y se utilizan para conectar diferentes 
    ubicaciones geográficas.

    Internetwork:
    Un internetwork es una colección de redes interconectadas. También se conoce como una red 
    de redes, puede incluir LAN, WAN y otras redes más pequeñas interconectadas para permitir 
    la comunicación y el intercambio de datos entre ellas. Algunas de estas redes interconectadas 
    pertenecen a grandes organizaciones públicas o privadas, como agencias gubernamentales o empresas 
    industriales.

    Internet:
    Internet es un ejemplo de un internetwork global que conecta millones de redes y dispositivos 
    en todo el mundo, es una red pública de escala mundial que utiliza el conjunto de protocolos TCP/IP 
    para permitir la comunicación y el intercambio de datos a nivel global.

    - Tanto las LAN como las WAN son tipos de redes que difieren en su alcance
      geográfico.
    - Tanto los internetworks como Internet implican la conexión de múltiples
      redes para permitir la comunicación y el intercambio de datos.

    - Las LAN son redes locales de alcance limitado, mientras que las WAN abarcan áreas más grandes.
    - Un internetwork se refiere a la conexión de múltiples redes, mientras que Internet es un 
      internetwork global que utiliza protocolos específicos.
    - Internet es un caso particular de internetwork que abarca todo el mundo,
      mientras que un internetwork puede ser más localizado.

3.  Un internetwork, también conocido como red de redes, se refiere a la conexión y la interacción 
    entre múltiples redes individuales. Estas redes pueden ser LANs (redes de área local) o WANs 
    (redes de área amplia) que están interconectadas utilizando dispositivos como enrutadores.
    Internet, por otro lado, es la red global de redes interconectadas que abarca todo el mundo. 
    Es una infraestructura masiva que permite la comunicación y el intercambio de información a nivel global.

4.  La tarjeta de interfaz de red (NIC) es un dispositivo de hardware que permite la conexión del 
    dispositivo a la red. 
    El puerto físico es el punto físico en un dispositivo donde se conecta un cable o un enchufe, 
    el cual se conecta con un host o con otro dispositivo de red.
    La interfaz de la red es la configuración lógica que permite que un dispositivo se comunique a 
    través de la red, también se podría decir que son puertos especializados de un dispositivo de 
    internetworking que se conecta con redes individuales.
